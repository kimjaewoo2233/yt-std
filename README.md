## Getting Started

#### install

```
npm i react-spinners@^0.13.8
npm i react-icons@^5.0.1
npm i tailwind-merge@^2.2.1
npm i zustand@^4.5.0

npm install next-themes # shadcn 다크모드위한 라이브러리
```

#### init

```
npx shadcn-ui@^0.8.0 init
 Default
 Slate
 yes

```
